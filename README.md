# jeffreyepstein

This is a simple redirect website that allows visitors to choose where to go: Discord, YouTube, Twitter, or Instagram.

## How to run

1. Start the server: `python3 -m http.server 8000`
2. Open your browser and go to `http://localhost:8000`

The website will display buttons linking to your social media profiles.

## Updating Bot Status

To update the bot status, edit the `status.json` file and change the "status" value to "online", "offline", or "fixing". The website will automatically reflect the change.
